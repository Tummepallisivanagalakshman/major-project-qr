# 🔳 QR Code Based Data Encoding & Decoding System

## 📌 Overview

This project is a QR Code–based application that allows users to generate and scan QR codes for storing and retrieving information. It provides a simple and efficient way to transfer data using images instead of manual input.

The system supports encoding user data into QR codes and decoding scanned QR codes to retrieve the original information.

---

## 🎯 Objective

The main objective of this project is:

* To simplify data sharing using QR codes
* To reduce manual data entry
* To demonstrate encoding and decoding techniques using Python

---

## 🚀 Features

* Generate QR code from user input
* Scan QR code using image or camera
* Decode QR code to retrieve original data
* Simple and user-friendly interface
* Fast and efficient data processing

---

## 🛠️ Technologies Used

* **Python**
* **Tkinter** – for GUI
* **OpenCV** – for camera-based scanning
* **Pyzbar** – for QR code decoding
* **qrcode library** – for QR code generation
* **PIL (Pillow)** – for image processing

---

## ⚙️ How It Works

1. User enters data (text/message)
2. System converts data into a QR code
3. QR code is saved as an image
4. User scans the QR code
5. System decodes and displays the original data

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Tummepallisivanagalakshman/major-project-qr.git
```

2. Navigate to the project folder:

```bash
cd major-project-qr
```

3. Install required libraries:

```bash
pip install -r requirements.txt
```

4. Run the project:

```bash
python main.py
```
